# Venera E-Ink

基于 [venera-app/venera](https://github.com/venera-app/venera) 修改的墨水屏优化版本。

## 修改内容

这个版本主要针对 e-ink 墨水屏设备做了优化：

- 增加 E-ink mode
- 将大量滑动列表改为分页浏览
- 尽量减少动画，降低低刷新率设备上的闪烁
- 为分页页面增加音量键翻页支持
- 优化搜索、收藏、发现、历史、图片收藏等页面在墨水屏上的体验

## 注意事项

- 本项目是基于 Venera 的 fork，不是继续维护的版本
- 不能直接作为原版 app 的覆盖更新安装
- 请先从原版导出/备份数据，再安装本版本并导入数据，即可继续正常使用
